About

MyBind is a free sysadmin friendly DNS hosting service. The interface is clean, simple and un-cluttered; focusing having useful features.

The underlying DNS server is ISC BIND, the most commonly used DNS server on the Internet (according to Wikipedia). MyBind has two geographically dispersed name servers (ns1.mybind.com and ns2.mybind.com), to provide standard DNS redundancy (RFC 2182).
